<br>
<br>
Tech stacks;
* Vue 2
* Vuex
* Axios
* SCSS
* json-server
* Vue Test Utils with Jest

## Project Setup and Run
**Project;**
<br>
npm install
<br>
npm run serve
<br>
<br>
**Local db;**
<br>
npm install -g json-server
<br>
json-server --watch db.json
<br>
http://localhost:3000/items
<br>
<br>
**Test;**
<br>
npm run test:unit
<br>
